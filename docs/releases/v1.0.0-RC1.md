# Release Notes — Career Workflow v1.0.0-RC1

**Release Date**: July 19, 2026  
**Baseline**: Production Release Candidate 1

---

## Executive Summary

Career Workflow **v1.0.0-RC1** marks the official feature-complete Release Candidate baseline for the core application orchestration engine, control plane API, and React operations console.

This release represents a clean, production-hardened system with 100% test coverage across 492 automated unit and integration tests.

---

## Release Highlights

1. **Dual Provider Job Acquisition**: Seamless job discovery blending authenticated Naukri API search with unauthenticated JobSpy multi-site scraping (Google Jobs, Indeed, LinkedIn).
2. **Local LLM Fit Scoring**: AI-driven match scoring using local oMLX / OpenAI-compatible endpoint evaluation against candidate tech stack and experience.
3. **Selection Budget & Diversity Constraints**: Capacity budget controls capping application rates and enforcing company/role-family diversity limits.
4. **Hybrid Questionnaire Resolution**: Deterministic rule matcher + local LLM form filler auto-resolving dynamic recruiter screening questions.
5. **Universal Job Link Preservation**: Raw job URLs and direct apply links preserved transactionally across acquisition, classification, selection, and terminal accounting.
6. **SQLite Application Ledger**: Transactional state store enforcing terminal accounting states (`APPLIED`, `REJECTED`, `EXPIRED`, `SKIPPED`).
7. **Intelligent Dual-Tier Caching**: Prompt-fingerprint caching for LLM scores and disk-backed search query acquisition caching.
8. **React Operations Console**: Enterprise web control plane providing real-time run execution, queue triage, and telemetry inspection.

---

## System Architecture Baseline

- **Language**: Python 3.10+
- **Control Plane**: FastAPI (`api/routes.py`)
- **Frontend**: React 18 + Vite + TailwindCSS (`frontend/`)
- **Database**: SQLite (`data/application_ledger.db`)
- **Scraper**: JobSpy (`python-jobspy`) & Selenium
- **AI Model Client**: `OMLXClient` (`src/llm/client.py`)

---

## Breaking Changes

- **Legacy UI Removal**: The legacy `career_ui_legacy` directory and NiceGUI components have been removed. All control plane operations are now routed through the React Operations Console and FastAPI Control Plane (`api/main.py`).
- **Dependencies**: Added required system libraries `psutil` and `pyyaml` to `requirements.txt`.

---

## Known Limitations

- **Scraper Anti-Bot Rate Limits**: Multi-site scraping via JobSpy depends on third-party job board availability. Using residential proxies (`JOBSPY_PROXIES`) is recommended for large multi-site scraping runs.
- **Local LLM Server Requirement**: AI job scoring and questionnaire resolution require a running local LLM server (e.g. oMLX / Ollama / vLLM on `http://127.0.0.1:8000/v1`).

---

## Upgrade & Quick Start Notes

1. Activate virtual environment and update dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Verify test suite:
   ```bash
   pytest
   ```
3. Start backend & frontend:
   ```bash
   uvicorn api.main:app --port 8000
   cd frontend && npm run dev
   ```
